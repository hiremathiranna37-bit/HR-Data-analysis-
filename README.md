# HR-Data-analysis-
This project focuses on HR data analysis using Microsoft Power BI to uncover workforce insights and support data-driven human resource decisions. The dashboard provides a clear view of employee demographics, attrition trends, and key HR performance indicators.

Overview
  - Analyzed HR data using Microsoft Power BI to derive workforce insights.
  - Cleaned and transformed data with Power Query.
  - Built DAX-based KPIs for attrition and employee metrics.
  - Designed an interactive HR dashboard with slicers and visuals.
  - Supported data-driven HR decisions through clear insights.

Tools
  - Power BI Desktop
  - Power Query
